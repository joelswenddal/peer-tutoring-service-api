Welcome to the Peer2Peer API Gateway!

This API allows registered Users to set up tutoring appointments with other students (who do not need to be registered).

  • Author: Joel Swenddal
  • Course: CS493 - Spring 2022
  • Oregon State University
  • Final Project
  • Home Address: https://final-peertutor-1215pm.uc.r.appspot.com
  • Project Overview:This project provides a Web API for a basic peer-to-peer tutoring service. This service allows Users to sign-up/login through a Google account, and then to use a JWT to authenticate when accessing the API. By signing up, Users can schedule tutoring appointments for themselves, which are then available for other students to book. Students, who do not need to sign up/authenticate, can enter basic information about themselves (a Student record) and then book appointments that have been made available by registered Users (Tutors).
  • Instructions: Click the button to be redirected to sign up as a User through a Google Account. Then, you will be able to make tutoring appointments with students using a JWT and an ID. These will allow you to access the Peer2Peer API. Note that if your JWT expires, you will need to login again to receive a new one.
